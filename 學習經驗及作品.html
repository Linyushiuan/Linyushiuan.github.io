BCIT第一到第三學期

在前三個學期中，我建立了完整的程式設計與計算機基礎。 我先透過 Python 學習程式設計的邏輯與基礎，再進一步學習 Java 與 物件導向程式設計，同時加強 數學邏輯能力。 此外，也修習了 HTML、CSS、JavaScript 的網頁開發課程，逐步累積前端開發的技能。

BCIT第四到第五學期

在第四與第五學期，我進一步學習了資料庫設計與SQL，了解一些關聯式資料庫的操作與規劃。 同時，在離散數學了解一些電腦運作的基本邏輯。透過 C 語言的程序式程式設計， 我更加理解程式的運作方式，並在系統分析與設計課程中，開始理解如何規劃與設計完整的軟體系統。 而Computer Architecture則幫我對硬體架構與軟體之間的關聯有了基本的概念。

BCIT第六學期(學習中)

目前我正在進一步學習 C 語言應用程式開發，透過更實作化的專案強化程式設計能力。 同時學習 UX/UI 基礎，了解以使用者為中心的設計原則；並學習 Agile 與 Scrum 方法， 培養敏捷開發與團隊協作的實務技巧。
